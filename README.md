# Brown v. Board of Education NHD Website

## Overview
This is a complete website template for a National History Day project about Brown v. Board of Education (1954). The site meets all NHD requirements for the website category and provides a comprehensive structure for presenting research on this landmark Supreme Court case.

## Files Included

### HTML Pages
- `index.html` - Home page with entry information and introduction
- `historical-context.html` - Background on Plessy v. Ferguson and Jim Crow era
- `the-case.html` - Details about the legal proceedings
- `key-figures.html` - Biographies of important people
- `decision-impact.html` - The ruling and immediate consequences
- `legacy.html` - Long-term significance and continuing impact
- `bibliography.html` - Annotated bibliography
- `process-paper.html` - Process paper explanation

### Styling
- `styles.css` - Complete stylesheet for professional appearance

## Tips for Success

### Research
- Start with primary sources (Court opinions, oral arguments)
- Use credible secondary sources (university presses, academic journals)
- Document everything immediately - don't lose sources!
- Take careful notes with page numbers for citations

### Writing
- Be clear and concise
- Use active voice
- Avoid repetition
- Make every word count (you have limited space!)

### Design
- Keep design clean and professional
- Ensure text is readable
- Use images purposefully, not decoratively
- Maintain consistent style throughout

### Citations
- Credit EVERY image and multimedia item
- Include both on-page credits and bibliography entries
- Remove clickable hyperlinks from displayed URLs
- Use proper citation format

## Common Mistakes to Avoid

1. **Exceeding word limit** - Count carefully and edit ruthlessly
2. **Missing image credits** - Every visual needs attribution
3. **Too much multimedia** - Stay under 3 minutes total
4. **Broken links** - Test everything before submitting
5. **File too large** - Compress images if over 100MB
6. **Editing during judging** - Site locks when submitted
7. **Teacher/school names** - Should not appear on site
8. **External links** - Don't link to outside websites (except required software)

## Technical Notes

### Image Optimization
- Save images as JPG (photos) or PNG (graphics)
- Resize to appropriate dimensions (max 1200px wide for hero images)
- Compress to reduce file size
- Name files descriptively (no spaces)

Before submitting, test your site in:
- Google Chrome

### Accessibility
- Ensure sufficient color contrast
- Use alt text for all images
- Test that site works without multimedia
- Verify navigation is clear
